# pyDraw


## Install Cmake

```bash
sudo apt install cmake
```

## Install SDL2

```bash
cd
git clone https://github.com/libsdl-org/SDL.git
cd SDL
cmake ./CMakeList.txt
sudo apt-get install libsdl2-gfx-dev
```
    
## Init Prog

```bash
git clone https://github.com/AntoninB0/pyDraw.git
cd ./PYDRAW
cd ./c
cmake ./CMakeLists.txt
```

## Run Prog

```bash
cd ./PYDRAW
python3 ./python/IDE.py
```
