# pyDraw


## Install SDL2

```bash
git clone https://github.com/libsdl-org/SDL.git
cd SDL
mkdir build
cd build
../configure
make
sudo make install 

sudo apt-get install libsdl2-gfx-dev
```
    
## Init Prog

```bash
cd c
cmake ./CMakeList.txt
```

## Run Prog

```bash
cd ./PYDRAW
python3 ./python/IDE.py
```
